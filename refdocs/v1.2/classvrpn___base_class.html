virtual void vrpn_BaseClass::mainloop ( )
pure virtual

Called once through each main loop iteration to handle updates.

Remote object mainloop() should call client_mainloop() and then call d_connection->mainloop(). Server object mainloop() should service the device and then call server_mainloop(), but should not normally call d_connection->mainloop().

Implemented in vrpn_Tracker_Remote, vrpn_Button_Remote, vrpn_Tracker_Server, vrpn_Button_PinchGlove, vrpn_Tracker_NULL, vrpn_Analog_Remote, vrpn_Tracker_Serial, vrpn_Button_Python, vrpn_Analog_Output_Remote, vrpn_Button_Example_Server, vrpn_Analog_Server, vrpn_Button_Server, vrpn_Analog_Output_Server, and omicron::vrpn_XInputGamepad.
